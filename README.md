## 游戏介绍
一款通过吹鼻涕泡把自己起飞的平台跳跃游戏.(一款不能跳跃的平台跳跃游戏[憋笑])

在本游戏里, 你将扮演"小鼻涕", 利用自己异于常人的鼻涕泡去冒险, 寻找自己的伙伴和同类. 最终击败BOSS---"无间地狱之堕, 于绝望中诞生的染血紫荆花, 最终燃尽千万世界的黄泉比量贩之安魂曲"拯救世界!

游戏支持Windows,Android双端, 加上最后BOSS关总共五关, 精美CG, 全程中文配音......
